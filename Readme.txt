Authors: Boqin Cai, Leo Eger, Bismark Ade, Ziyang Dai, Ziyan Ding, Yumeng Zhou

Basically, the coronamap folder is the website of Leo Eger, Bismark Ade, Ziyang Dai, Ziyan Ding, Yumeng Zhou. And the location sharing part is developed by Boqin Cai.

It is standalone static website. Finally, the corona map website is integrated in the framework of Boqin Cai's website.

* The index.html is the default page of our project. It is a page with coronamap.

* The locationlogin.html is the login page of location sharing.

* The realmap.html is the user interface of location sharing.

* The script files in PyScripts folder are the backend of python Flask.
